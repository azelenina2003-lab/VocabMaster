# VocabMaster – приложение для запоминания слов и терминов

VocabMaster — это веб-приложение на Django, которое помогает пользователям запоминать иностранные слова, термины и определения. Пользователь может создавать собственные категории (колоды) карточек, добавлять в них термины с определениями, изучать их в режиме переворота и проходить тестирование с автоматической проверкой. Категории (колоды) можно редактировать, удалять, те же функции доступны и для карточек со словами и определениями. 

## Основные возможности
- Регистрация и аутентификация пользователей.
- Создание, редактирование и удаление категорий.
- Добавление, редактирование и удаление записей (термин + определение).
- Режим изучения: определение демонстрируется при нажатии кнопки "показать определение"
- Режим тестирования: ввод определения для термина с проверкой правильности.
- Для каждого термина, подсчитывается статистика правильных ответов.
- Полная изоляция данных: каждый пользователь видит только свои категории и записи.

## Технологический стек
- **Python 3.13**
- **Django 6.0.3**
- **Bootstrap 5** (для адаптивного интерфейса)
- **SQLite** (база данных по умолчанию)
- **pytest**, **pytest-django**, **factory-boy** (для тестирования)
